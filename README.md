## Census Affiliate Nextjs

## Prerequisite
- Node installed

## Setup
```bash
# install yarn
npm instal -g yarn
# 2. create a file called .env on the root of this folder and copy the content from below

## install packages 
yarn install

## run application
yarn dev

## Application will run in http://localhost:3000/
```

## Env file
```
NEXT_PUBLIC_API_URL=http://localhost:5000
NEXT_PUBLIC_CENSUS_URL=https://censustravel.com/
```
